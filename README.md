## Description

Docker-compose setup for starting [MediaWiki](https://www.mediawiki.org).

## Setup

```sh
cd /opt
git clone https://github.com/solution-libre/docker-mediawiki.git mediawiki
cd mediawiki
```

Change the hostname value in `.env`.

## Usage

```sh
cd /opt/mediawiki
docker-compose up -d
```
